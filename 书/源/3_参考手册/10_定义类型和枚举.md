# 定义类型和枚举


## 定义类型

关键词 `定义类型`。

自定义数据类型 (结构体)。
类似 rust 的 `struct`。

格式:

```
定义类型名称, 泛型列表:
  成员名称(类型)。
```

其中泛型是可选的, 比如:

```
定义类型吃货:
  名称(文本)。
  身高(数字)。
  体重(数字)。
```


## 定义枚举

关键词 `定义枚举`。

枚举数据类型。
类似 rust 的 `enum`。

格式:

```
定义枚举名称, 泛型列表:
  枚举项名称(类型)。
```

其中类型是可选的, 比如:

```
定义枚举可选, 泛型T:
  无。
  有(T)。
```

不使用泛型:

```
定义枚举水果:
  苹果。
  香蕉。
  橘子。
  西瓜。
```


## 类型别名

格式:

```
定义类型名称, 原类型。
```

比如:

```
定义类型二, u1。
```
